About

This data is just a mirror of data from ultimateframedata.com, made availiable through an API with a simple frontend.
I did not gather this data, it is scraped from ultimateframedata using a series of simple python scripts you can find in the root of the api's github. API methods are documented here (with the root url cade.schlaefli.dev) but only GET requests for non-auth methods cross site enabled for security reasons.

The code for this frontend is availiable here, and the api is avaliable here. I've also done a couple game jams with a small team you can check out here
